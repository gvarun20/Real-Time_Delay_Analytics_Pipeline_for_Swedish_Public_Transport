# Real-Time Delay Analytics Pipeline for Swedish Public Transport

An end-to-end data engineering project: Airflow-orchestrated GTFS ingestion, PySpark transformation, Kimball star schema in PostgreSQL, and a Streamlit dashboard.

**Status:** Planning phase — see the full 4-week build plan below.
